CSS - The Box Model

Cheat Sheet

Box Model Item Property Description
Content Height The height of the content is set in a measurable data such as pixels or can be set by pertentages.
Width The width of the content is set in a measurable data such as pixels or can be set by pertentages.
Padding padding Shorthand padding e.g. padding: 0 10px 20px 0; is the quickest way to add padding to more than one side. The values in order are 1. top, 2. right, 3. bottom and 4. left.
padding-top
padding-right
padding-bottom
padding-left
Border border Shorthand border e.g. border: 1px solid green; is the quickest way to add a border. The values in order are: 1. width 2. style and 3. color.
border-width
border-style
border-color
Margin margin Shorthand margin e.g. margin: 0 10px 20px 0; is the quickest way to add margin to more than one side. The values in order are 1. top, 2. right, 3. bottom and 4. left.
margin-top
margin-right
margin-bottom
margin-left

Box Model inner to outer:

  1. Content - Starting with the inner of the box model is the content.
  2. Padding
  3. Border
  4. Margin
